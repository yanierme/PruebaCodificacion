# Conexion a base de datos Postgres
 
La aplicación fue construida con las siguientes herramientas y lenguaje de programación:

* Java 8
* Gradle 4.10
* libreria JDBC
* postgresql

Para la creación del proyecto se creó una base de datos postgres de forma local
con el nombre "empleados"

Para ejecutarlo en otra maquina se debe crear la base de datos postgres con la 
estructura espeficada en la prueba enviada.
